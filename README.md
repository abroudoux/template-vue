## Template-Vue

Vue Template for my future projects with :
- SASS
- TypeScript
- Babel
- Router
- VueUX
- Prettier
- ESlint
- Font Awesome

# If you want to use it : 

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
